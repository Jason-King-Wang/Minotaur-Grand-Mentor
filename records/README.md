# Records

This folder stores timestamped Markdown records generated by `tools/log_event.py`.

Expected structure:

```text
records/
  YYYY-MM-DD/
    HHMMSS-phase-name.md
```

Rules:

- Do not delete existing records.
- Do not overwrite old records.
- Each meaningful Codex execution or project phase should leave a record.
- Records should describe what changed, why, and what comes next.
