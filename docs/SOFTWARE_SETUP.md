# Software Setup

## Codex / Engineering

- Git: version control and GitHub upload.
- Python 3.10+: logging, validation, public data builder, tests, placeholder generation.
- Browser: dashboard and overlay preview.
- OBS Studio: Browser Source for PNGTuber overlay.
- Obsidian: project notes and local knowledge base.

## Art Production

- Photoshop, Clip Studio Paint, Krita, or equivalent: final PNGTuber art and layered PSD.
- Transparent PNG export support.
- Consistent canvas size: `2048x2048` or `3000x3000`.

## Live2D Rigging

- Live2D Cubism Editor: rigging, mesh, deformers, physics, expressions.
- VTube Studio: model testing and streaming.
- Optional browser Live2D runtime later, after the model is real.

## Notes

Codex can prepare file structures, specs, checklists, and static web tooling. Codex cannot complete production Live2D rigging without final art and a layered PSD.
