# Style Guide

## Core Palette

- Black: leather, shadows, serious trading desk mood.
- Gold: eyes, tech lines, mentor authority, value signal.
- Deep blue: trading screens, AI systems, night-market atmosphere.
- Brown: minotaur skin and natural creature identity.

## Visual Style

- Cyber trading mentor.
- Semi-realistic VTuber.
- Clean silhouette.
- Strong upper-body read.
- Blue-gold financial dashboard glow.
- Original character design, not copied from references.

## Composition Requirements

- Half-body or bust composition.
- Character centered.
- Horns fully visible.
- Face readable at small sizes.
- Mouth shape clear enough for PNGTuber switching.
- Transparent background for avatar assets.
- Trading motifs may appear as abstract chart light, grid glow, or interface atmosphere.

## Expression Requirements

- `idle_closed`: calm, dominant, mouth closed.
- `talk_open`: clear mouth-open speaking pose.
- `blink_closed`: eyes closed, same pose as idle.
- `happy`: confident mentor smile.
- Optional expressions should stay on-model.

## Forbidden

- Horror gore.
- Excessive realism that breaks VTuber readability.
- Cropped horns.
- Background text.
- Logos.
- Watermarks.
- Direct imitation of reference images.
- Investment-advice claims.
