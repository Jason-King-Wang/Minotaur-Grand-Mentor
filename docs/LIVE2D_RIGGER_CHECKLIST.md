# Live2D Rigger Checklist

## Required Before Rigging

- [ ] Final layered model PSD exists.
- [ ] PSD is not flattened.
- [ ] All major layers are named clearly.
- [ ] Horns are fully visible and separated.
- [ ] Eyes are separated into white, iris, pupil, highlight, upper eyelid, lower eyelid.
- [ ] Mouth has inner mouth, upper lip, lower lip, teeth, and tongue.
- [ ] Jacket body, collars, zipper, and gold trims are separated.
- [ ] No watermark.
- [ ] No background text or logos.
- [ ] Character matches Minotaur Grand Mentor design.

## Rigging Expectations

- Head XYZ movement.
- Eye tracking and blinking.
- Mouth open/close and vowel shapes.
- Brow movement.
- Horn-safe head rotation.
- Breathing motion.
- Jacket secondary motion.
- Optional hand/pointing motion if final art supports it.

## Delivery

- Cubism project file.
- `.moc3`.
- `model3.json`.
- Texture files.
- Physics and expressions when applicable.
- Test notes for VTube Studio.
